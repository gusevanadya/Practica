🔗 3. JOIN (связи)

💡 Подсказка по типам JOIN
INNER JOIN
📊 Возвращает только те строки, где есть совпадение в ОБЕИХ таблицах
Пример: Найти всех сотрудников, у которых есть отдел (исключит сотрудников без отдела)
LEFT JOIN
📋 Возвращает ВСЕ строки из левой таблицы + совпадения из правой (если нет — NULL)
Пример: Все сотрудники (даже без отдела) + информация об отделе, если есть
RIGHT JOIN
📌 Возвращает ВСЕ строки из правой таблицы + совпадения из левой (если нет — NULL)
Пример: Все отделы + сотрудники в них (отделы без сотрудников тоже попадут)
FULL JOIN
🔄 Возвращает ВСЕ строки из обеих таблиц (совпадения + несовпадения с NULL)
Пример: Все сотрудники и все отделы, даже если нет соответствий
